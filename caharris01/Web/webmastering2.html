Web Mastering 20-21

Web Mastering is Peak to Peak's introductory course in webpage design and web-based computer programming. This class is appropriate for students who want an introduction to how to make computers perform tasks: whether for scientific computing or interactive visual presentation.

Teacher:

Mr. Robert Hettmansperger
robert.hettmansperger@bvsd.org
Room N-130

Goals:

HTML Use HTML Tags to structure a web page
CSS Use CSS rules to stylize a web page
JavaScript Use JavaScript to make pages responsive to events
  • Create and use variables
  • Prompt the user for input
  • Perform actions conditionally
  • Perform actions repeatedly using loops
  • Perform actions repeatedly using timers
  • Write functions to organize and structure programs as eventresponders
  • Draw basic figures to an HTML canvas
